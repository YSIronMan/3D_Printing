# 3D_Printing
丰富的 3D 打印机资料，包含图纸、模型、上位机程序等

## dbot

CoreXY 结构，自主设计的 3D 打印机

## kossel_800

三角洲 结构，简称 k800，非标准打印件的 3D 打印机

## prusa_i3

最简单的 XYZ 结构，简称 i3，适合入门玩家的 3D 打印机

## ultimaker2_al

铝型材版 UM2 打印机，仿照商业机器 UM2 图纸设计的 3D 打印机

## 打印模型

一堆 3D 打印模型，大部分经过打印测试，欢迎使用~
